Entendendo o Loop for...of em JavaScript

O loop for...of em JavaScript é usado para iterar sobre elementos de estruturas de dados iteráveis, como arrays, strings, Map, Set e outros.

Ele funciona atribuindo o próximo valor da estrutura iterada a uma variável especificada em cada iteração, ao contrário do for...in, que itera sobre as chaves ou índices. O for...of itera diretamente sobre os valores.

Sintaxe

for (variável of iterável) {
    // Código a ser executado
}

Exemplo com Arrays

No exemplo abaixo, o loop percorre um array e imprime cada valor:

const array = ['a', 'b', 'c'];
for (let value of array) {
    console.log(value);
}

Saída:

a
b
c

Exemplo com Strings

O for...of também pode ser utilizado para iterar sobre cada caractere de uma string:

const string = "Hello";
for (let char of string) {
    console.log(char);
}

Saída:

H
e
l
l
o

Diferença entre for...of e for...in

  • for...in: Itera sobre as propriedades (índices ou chaves) de um objeto ou array.
  • for...of: Itera sobre os valores diretamente.
