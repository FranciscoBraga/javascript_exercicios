Concatenação de Arrays em JavaScript

1. Usando o Método concat()

O método concat() junta dois ou mais arrays e retorna um novo array, sem modificar os arrays originais.

const array1 = [1, 2, 3];
const array2 = [4, 5, 6];

const novoArray = array1.concat(array2);

console.log(novoArray); // [1, 2, 3, 4, 5, 6]

2. Usando o Operador Spread (...)

O operador spread espalha os elementos de um array, permitindo concatenar arrays de forma concisa.

const array1 = [1, 2, 3];
const array2 = [4, 5, 6];

const novoArray = [...array1, ...array2];

console.log(novoArray); // [1, 2, 3, 4, 5, 6]

3. Usando o Método push() com o Operador Spread

O método push() adiciona elementos ao final de um array. Quando usado com o operador spread, ele concatena arrays modificando o array original.

const array1 = [1, 2, 3];
const array2 = [4, 5, 6];

array1.push(...array2);

console.log(array1); // [1, 2, 3, 4, 5, 6]

4. Usando um Loop for

Você pode concatenar arrays iterando sobre um array e adicionando seus elementos a outro array usando um loop for.

const array1 = [1, 2, 3];
const array2 = [4, 5, 6];

for (let i = 0; i < array2.length; i++) {
    array1.push(array2[i]);
}

console.log(array1); // [1, 2, 3, 4, 5, 6]

5. Usando o Método reduce()

O método reduce() pode ser usado para concatenar arrays, especialmente se você tiver um array de arrays.

const arrays = [[1, 2], [3, 4], [5, 6]];

const concatenado = arrays.reduce((acc, curr) => acc.concat(curr), []);

console.log(concatenado); // [1, 2, 3, 4, 5, 6]

6. Usando o Método flat()

O método flat() "achata" arrays aninhados em um único array, útil quando se tem arrays dentro de arrays.

const arrays = [[1, 2], [3, 4], [5, 6]];

const concatenado = arrays.flat();

console.log(concatenado); // [1, 2, 3, 4, 5, 6]
