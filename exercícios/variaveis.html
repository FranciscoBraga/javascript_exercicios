Variáveis em JavaScript

Variáveis em JavaScript são identificadores usados para armazenar dados que podem ser manipulados e referenciados ao longo do código. Elas são fundamentais para a programação, pois permitem o armazenamento de valores temporários, a execução de operações matemáticas, a manipulação de strings e a tomada de decisões com base em condições.

Declaração de Variáveis em JavaScript

JavaScript permite a declaração de variáveis utilizando três palavras-chave principais:

  1. var: É a forma mais antiga de declarar variáveis. Uma variável declarada com var tem escopo de função ou global, dependendo de onde é declarada. Não respeita o escopo de bloco.
    2. var nome = "João";
  2. let: Introduzida no ES6 (ECMAScript 2015), let permite a declaração de variáveis que respeitam o escopo de bloco. É preferida em relação ao var para evitar problemas de escopo.
    3. let idade = 30;
  3. const: Também introduzida no ES6, const declara variáveis que não podem ser reatribuídas após sua inicialização. Assim como let, const respeita o escopo de bloco.
    4. const PI = 3.14;

Tipos de Variáveis em JavaScript

JavaScript é uma linguagem de tipagem dinâmica, o que significa que uma variável pode ser inicializada com um tipo de dado e, posteriormente, conter um tipo diferente. Os tipos de dados em JavaScript podem ser divididos em duas categorias principais: tipos primitivos e tipos de referência.

Tipos Primitivos

  • string: Cadeia de caracteres. 
    let nome = "Maria";
  • number: Números, tanto inteiros quanto decimais. 
    let idade = 25;
let temperatura = -5.3;
  • boolean: Representa um valor lógico: true ou false. 
    let isAluno = true;
  • undefined: Uma variável que foi declarada, mas não inicializada, possui o valor undefined. 
    let valor;
console.log(valor); // undefined
  • null: Representa intencionalmente um valor vazio ou inexistente. 
    let obj = null;
  • symbol: Introduzido no ES6, Symbol é um valor único e imutável que pode ser usado como identificador para propriedades de objetos. 
    let id = Symbol('id');
  • bigint: Representa números inteiros de precisão arbitrária. Útil para cálculos com números maiores do que o tipo number pode suportar. 
    let bigNumber = BigInt(9007199254740991);

Tipos de Referência

  • object: Uma coleção de propriedades, onde cada propriedade é um par chave/valor. Pode ser um objeto literal, array, função, etc. 
    let pessoa = {
    nome: "João",
    idade: 30
};
  • array: Um tipo de objeto que armazena uma coleção de elementos indexados. 
    let frutas = ["maçã", "banana", "laranja"];
  • function: Em JavaScript, funções são objetos de primeira classe, o que significa que podem ser armazenadas em variáveis, passadas como argumentos para outras funções, etc. 
    function saudacao() {
    console.log("Olá!");
}

Como Usar Variáveis em JavaScript

  1. Declaração e Inicialização: Você declara uma variável usando var, let ou const, e pode inicializá-la com um valor. 
    let cidade = "São Paulo";
const ano = 2024;
var idade = 28;
  2. Reatribuição: Variáveis declaradas com var ou let podem ser reatribuídas. Variáveis declaradas com const não podem ser reatribuídas. 
    let contador = 1;
contador = 2; // válido

const PI = 3.14;
// PI = 3.1415; // inválido, causa um erro
  3. Escopo de Variáveis: Variáveis declaradas com var têm escopo global ou de função. Variáveis declaradas com let e const têm escopo de bloco. 
    if (true) {
    var x = 5; // escopo global
    let y = 10; // escopo de bloco
    const z = 15; // escopo de bloco
}
console.log(x); // 5
// console.log(y); // ReferenceError: y is not defined
// console.log(z); // ReferenceError: z is not defined

Boas Práticas

  • Use let e const em vez de var para evitar problemas de escopo.
  • Prefira const a let quando não precisar reatribuir o valor, para deixar claro que a variável não deve mudar.
  • Escolha nomes de variáveis descritivos para melhorar a legibilidade do código.
